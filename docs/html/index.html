User Interface Node

This module implements an user interface to allow the user to select the desired driving mode.

Subscribes to:

  • /automatic/cmd_vel

  • /manual/cmd_vel

  • /assisted/cmd_vel

  • /move_base/action_topics

Publishes to:

  • /cmd_vel

scripts.user_interface.astd_vel_callback(msg)

Callback function that is called each time that a new message on the “/assisted/cmd_vel” topic is published. The function is used to published it on the “/cmd_vel” topic when the user has selected the corresponding mode.

Parameters

msg (Twist message;) – contains the velocity generated by the “collision_avoidance” node;

Returns

none;

Collision Avoidance Node

This module implements a controller that update the robot’s velocity in order to avoid crashing into obstacles inside the map.

Subscribes to:

  • /scan

  • /manual/cmd_vel

Publishes to:

  • /assisted/cmd_vel

scripts.collision_avoidance.astd_vel_callback(msg)

Function that is called each time that a new message on the /manual/cmd_vel topic is published. The function is used to copy such message in another one of the same type to be published on the /assisted/cmd_vel topic.

Parameters

msg (Twist message) – contains the velocity generated by the teleop_twist_keyboard node;

Returns

none;

scripts.collision_avoidance.avoid_collision(regions)

Function that is used to implement the collision avoidance mechanism. It checks the distances of the robot from the external environment and updates its velocity in order to avoid collisions.

Parameters

regions (dictionary direction:value) – contains the minimum distances in 5 direction;

Returns

none;

scripts.collision_avoidance.main()

This function initializes the ROS node.

scripts.collision_avoidance.scan_callback(msg)

Function that is called each time that a new message on the /scan topic is published. It divides the ranges vector into five regions in order to get the minimum value in each and give this as input to the collision avoidance function.

Parameters

msg (LaserScan message) – contains the ranges returned by the sensor;

Returns

none;
